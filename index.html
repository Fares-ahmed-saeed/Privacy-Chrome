تحسين تجربة المستخدم – UX Enhancer Pro

إضافة UX Enhancer Pro مصممة لتحسين تجربة المستخدم عند تصفح صفحات الويب من خلال أدوات تخصيص العرض مثل التكبير، وضع القراءة، المؤشر المخصص، الخلفيات، والمكبر. نحن نأخذ خصوصيتك على محمل الجد، وهذه الصفحة تشرح بالضبط كيف يتم التعامل مع بياناتك.

1. جمع البيانات

هذه الإضافة لا تجمع أي بيانات شخصية أو معلومات عن المستخدمين، ولا تقوم بتتبع نشاط التصفح، ولا تقرأ محتوى الصفحات لأغراض تخزين أو إرسال.

جميع الإعدادات مثل:

  • حجم التكبير
  • وضع القراءة
  • إعدادات الخط
  • الخلفية
  • الملفات الشخصية (Profiles)

يتم حفظها محليًا داخل متصفح المستخدم فقط باستخدام Chrome Storage، ولا يتم إرسالها إلى أي خادم أو جهة خارجية.

2. استخدام الأذونات

تستخدم الإضافة الأذونات التالية فقط لتحقيق وظيفتها:

  • storage – لحفظ إعدادات المستخدم محليًا.
  • activeTab – للوصول المؤقت إلى التبويب الحالي عند تفاعل المستخدم مع الإضافة.
  • scripting – لتطبيق ميزات مثل وضع القراءة والتكبير والمؤشر داخل الصفحة.
  • host permissions – للسماح بتعديل عرض الصفحات التي يختارها المستخدم.

لا يتم استخدام هذه الأذونات للوصول إلى أي بيانات خاصة أو إرسال أي معلومات خارج جهاز المستخدم.

3. مشاركة البيانات

نحن لا نبيع أو نشارك أو ننقل أي بيانات إلى أطراف ثالثة لا يتم إرسال أي معلومات إلى خوادم خارجية

4. الأمان

جميع عمليات الإضافة تتم داخل متصفح المستخدم فقط لا يتم تحميل أو تشغيل أي كود خارجي أو سكربتات من الإنترنت

5. التغييرات على السياسة

إذا تم تحديث هذه السياسة في المستقبل، سيتم نشر النسخة الجديدة في نفس صفحة سياسة الخصوصية.

6. التواصل

إذا كان لديك أي أسئلة بخصوص هذه السياسة أو الإضافة، يمكنك التواصل مع مطور الإضافة عبر صفحة المتجر.
